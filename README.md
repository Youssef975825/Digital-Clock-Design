# Digital & Analog Clock

A sleek, interactive clock project featuring both analog and digital displays, built with modern web technologies. This project utilizes a Neumorphic design aesthetic to provide a clean and professional user interface[cite: 2].

**[🔗 Click here to view the Live Demo](YOUR_VERCEL_LINK_HERE)**

## 🚀 Features

- **Dual Display:** Shows time through both smooth-rotating analog hands and a real-time digital clock[cite: 1].
- **Neumorphic Design:** High-quality visual style achieved with advanced CSS shadows and layered design principles[cite: 2].
- **Responsive Layout:** The design is built to be flexible and look great on various screen sizes[cite: 2].
- **Dynamic Updates:** Uses JavaScript to sync both the analog hands and the digital text with the user's system time[cite: 1].

## 🛠 Technologies Used

- **HTML5:** Semantic structure for the clock and display components[cite: 3].
- **CSS3:** Styling, animations, and Neumorphic effects[cite: 2].
- **JavaScript:** Real-time logic for time calculations and DOM manipulation[cite: 1].

## 📂 Project Structure

- `index.html`: The main markup file[cite: 3].
- `style.css`: Contains all visual styles, animations, and layout logic[cite: 2].
- `main.js`: Handles the core logic and timing updates[cite: 1].

## 🚀 Deployment

This project is optimized for static deployment. You can easily deploy it using **Vercel**:

1. Push your code to a **GitHub** repository.
2. Log in to [Vercel](https://vercel.com/) and select **Add New Project**.
3. Import your GitHub repository.
4. Click **Deploy** and Vercel will handle the rest!

## 💡 Customization

The project uses CSS variables (`--clr`) to manage the colors for the clock hands and digits. You can quickly change the theme colors directly in the `index.html` file[cite: 2, 3].
